контрольные вопросы:
1.Функция — именованный блок кода, выполняющий конкретную задачу. Преимущества: повторное использование, структурирование кода, изоляция логики, упрощение отладки.

2.Модуль — файл с расширением .py. Пакет — каталог с модулями, содержащий файл __init__.py.

3.Создание пакета: создать папку с именем пакета, добавить в неё __init__.py и модули, при необходимости описать структуру в setup.py.

4.Импорт — подключение модуля/пакета. Способы: import module, import module as alias, from module import name.

5.Обработка исключений: используется try-except-else-finally. При возникновении исключения в try управление передаётся в соответствующий except.

6.Рекурсивная функция — вызывает саму себя. Полезна для задач с рекурсивной структурой: обход деревьев, факториал, алгоритмы "разделяй и властвуй".

7.Область видимости определяет, где доступна переменная. Уровни: локальная (в функции), объемлющая (внешняя функция), глобальная (модуль), встроенная (built-in).

8.Передача аргументов: по умолчанию — передача по ссылке (объекта). Аргументы могут быть позиционными, именованными, с значениями по умолчанию, переменного количества (*args, **kwargs).

9.Декораторы — функции, изменяющие поведение других функций без изменения их кода. Используются для добавления логирования, проверок, кэширования, измерения времени выполнения.
